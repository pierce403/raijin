raijin.sh

Ephemeral shell access for troubleshooting the systems you control.

Start a temporary session, paste a one-line Python bootstrap into a Linux VM or container you administer, and work in a live browser terminal. No daemon. No install residue. No persistence by default.

Warning: this grants remote shell access to the current environment. Use it only on systems you control.
